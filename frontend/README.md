## Currently implemented:
- [x] Filter Button (unused as I use useEffect for the selection.)
- [x] Add Task Button

- [x] Delete All Button
- [x] Delete Button
- [x] Save Button

## Setup
1. Open terminal in the current working directory.
2. Type `pnpm install`
3. Change configuration of API path in `config/axios.tsx` (optional)

## Running
- Run the client with `pnpm run dev` command.
